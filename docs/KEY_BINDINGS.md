# Key Bindings

| Key        | Action                       |
|------------|------------------------------|
| ↑/↓        | Move selection               |
| PgUp/PgDn  | Page navigation              |
| Home/End   | Jump top/bottom              |
| `/`        | Filter prompt                |
| **F**      | Clear current filter         |
| `?`        | Help popup                   |
| Enter      | Open session in browser      |
| q / Esc    | Quit                         |
|------------|------------------------------|
