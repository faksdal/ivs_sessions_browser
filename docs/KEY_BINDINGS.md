# Key Bindings

| Key          | Action                         |
|--------------|--------------------------------|
| ↑/↓          | Move selection                 |
| PgUp/PgDn    | Page navigation                |
| Home/End     | Jump top/bottom                |
| `/`          | Filter prompt                  |
| **C**        | Clear current filter           |
| **T**        | Go to today's date             |
| `?`          | Help popup                     |
| Enter        | Open session in browser        |
| q / Q        | Quit                           |
| ------------ | ------------------------------ |
